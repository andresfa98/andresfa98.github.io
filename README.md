# Portafolio Web (GitHub Pages)

## 1) Crear cuenta de GitHub
- Ve a https://github.com/signup
- Crea tu usuario.

## 2) Crear repo para la web
- Nombre recomendado: `tuusuario.github.io`
- Público.

## 3) Subir estos archivos
Sube `index.html` y `styles.css` a la raíz del repo.

## 4) Publicar
Si el repo se llama `tuusuario.github.io`, se publica automáticamente en:
- `https://tuusuario.github.io`

Si usas otro nombre de repo:
1. Ve a `Settings` -> `Pages`
2. En `Build and deployment`, elige `Deploy from a branch`
3. Branch: `main`, carpeta `/root`
4. URL final: `https://tuusuario.github.io/nombre-repo`

## 5) Añadir proyectos
En `index.html`, duplica un bloque `<article class="card">` por cada proyecto y cambia:
- título
- descripción
- links demo/código
